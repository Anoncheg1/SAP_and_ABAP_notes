***ѕрименение**4)*
When open SQL ABAP statements apply to a client-dependent table, SAP automatically adds the client selection in the WHERE clause.

For example SELECT * FROM mara WHERE matnr LIKE '1%' will be automatically translated into SELECT * FROM mara WHERE mandt = sy-mandt AND matnr LIKE '1%' (where sy-mandt is the client where the program runs).

If you try to do client selection yourself, it is ignored, except if you add the CLIENT SPECIFIED keyword.
